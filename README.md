# Guía de ejercicios - Patrones de diseño en PHP

Contenido:
- ejercicio1_factory/: implementación del patrón Factory para crear personajes (Esqueleto, Zombi).
- ejercicio2_adapter/: implementación del patrón Adapter para compatibilidad de archivos Windows7 -> Windows10.
- ejercicio3_decorator/: implementación del patrón Decorator para agregar armas a personajes.
- ejercicio4_strategy/: implementación del patrón Strategy para diferentes salidas (consola, JSON, TXT).

Cada ejercicio incluye un archivo ejecutable desde CLI (php) que muestra ejemplos de uso.

Instrucciones rápidas:
clona el repo 
2. Desde la terminal ejecuta,
   `php run_factory.php` 
   `php run_adapter.php`
   `php run_decorator.php`
   `php run_strategy.php`
